== Description ==

A plugin to display your Steam Gameplay statistics in any page or post. Simply add the appropriate shortcode to a page or post, and you're ready to go!

This includes all Steam based games, including Team Fortress 2, Left 4 Dead and Left 4 Dead 2,  Counter Strike, and more!

Do you speak a language other than English? Do you have time to translate some strings? If so, I'd love to [hear from you](http://pento.net/contact/)!

== Installation ==

= The Good Way =

1. In your WordPress Admin, go to the Add New Plugins page
1. Search for: steam stats
1. Steam Stats should be the first result. Click the Install link.

= The Old Way =

If you use this method, please remember to Deactivate/Activate Steam Stats between upgrades, to ensure that the upgrade routines are run properly.

1. Upload the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

= The Living-On-The-Edge Way =
(Please don't do this in production, you will almost certainly break something!)

1. Checkout the current development version from http://wordpress-steam-stats.googlecode.com/svn/trunk/
1. Subscribe to the [update feed](http://code.google.com/p/wordpress-steam-stats/source/list) to be notified of changes
